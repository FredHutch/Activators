Here I follow Jacob's protocols and advices and try to observe trends and obtain preliminary conclussions about what makes a sequence a TAD.

input is composed of 30-mers from a sorted library ('ARG3'), splited into ~20000 positives and ~20000 negatives.
GOAL: Undestand how composition of aminoacids and/or other calculated or predicted physico-chemical properties of 30mers contribute to define a Transcription Activator Domain or TAD (positives in our data-set - negatives are non-TADs)

# Since I will do the same for regression and classification, I make it a function
def split_data(data, selection=aa, column_label_score_NAME='scores', seed=0):
    train, test = (np.array([0.8, 0.1]) * data.shape[0]).astype(int)
    
    X, y = data[selection].values, data[column_label_score_NAME].values
    
    np.random.seed = seed
    idx = np.arange(data.shape[0])
    np.random.shuffle(idx)
    
    X, y = X[idx], y[idx]
    
    Xtrain, ytrain = X[:train], y[:train]
    Xtest, ytest = X[train:-test], y[train:-test]
    Xvalid, yvalid = X[-test:], y[-test:] 
   
    return(Xtrain, ytrain, Xtest, ytest, Xvalid, yvalid)

# create two sets of train, test, validation, one for regression and one for classification.
# These are lists that consists of 6 elements (3 sets of X,y): (X,y train), (X,y test) and (X,y validation) 
# e.g. regression[0] = X_train, regression[5]= y_validation
regression = split_data(data, aa, 'scores')
classification = split_data(data_classif, aa, 'TAD')

def roc_curve(X,y):
    # make sure X and y are numpy arrays
    X,y = np.array(X), np.array(y)
    # thresholds to use for defining fpr and tpr
    scale = np.linspace(X.min(), X.max(), 50)
    # define fpr and tpr functions to use with different thresholds
    def tpr(y_hat, y):
        a,b = set(np.hstack(np.where(y==1))), set(np.hstack(np.where(y_hat==1)))
        return(len(a & b) / len(a))
    def fpr(y_hat, y):
        a,b = set(np.hstack(np.where(y==0))), set(np.hstack(np.where(y_hat==1)))
        return(len(a & b) / len(a))
    # Core of the function where TPR and FPR are calculated
    TPR, FPR = np.zeros(50), np.zeros(50)
    for n,s in enumerate(scale):
        # threshold of variable defines what is + and -
        y_hat = np.array([1 if x>=s else 0 for x in X])
        TPR[n] = tpr(y_hat, y)
        FPR[n] = fpr(y_hat, y)
    # the resulting array will contain thresholds, fpr and tpr
    roc = np.vstack([scale, FPR, TPR])
    return(roc)

import itertools

# list all "k"-mers that exist in the sequence "seq"
def kmerize(seq, k):
    return ' '.join(seq[i:i+k] for i in range(len(seq) -k +1))

# generate all possible k-mers from "aa" list 
def kmers(k):
    return [''.join(kmers) for kmers in itertools.product(aa, repeat=k)]

# generate one column for each of all possible kmers in data (k=2 here)
for kmer in kmers(2):
    data[kmer] = [kmerize(seq, 2).count(kmer) / 29. for seq  in data.index]

Deep learning predictions ¶

I will start with a Dense network which is fast to train and then I will train on the gizmo server a convolutional network and a convolutional + recurrent network with the hope to incorporate in the predictive model:

  • A combination of 2-mers in defined positions or distances from each-other
  • A motif Later I could dig into the models and find these features.

I will optimize the number of layers and the number of nodes per layer in a dense network (multi-layer perceptrons).

I will train a regression (fisrt) and a classification model. The regression model includes a higher number of samples but the scores are empirical and will include a error that is unknown to me (I only have 18 experimentally validated samples to figure it out). The classification model includes 2/3 of the data used for regression but I am more confident on the labeling.
